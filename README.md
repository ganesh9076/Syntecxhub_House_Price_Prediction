# House Price Prediction

## Overview
Machine Learning project to predict house prices using housing features.

## Features
- Data Preprocessing
- Model Comparison
- Linear Regression
- Random Forest
- Feature Importance Analysis
- Correlation Heatmap
- Actual vs Predicted Visualization
- Dynamic User Input Prediction

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
